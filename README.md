## 🚀 Tech Stack
- **Frontend Framework:** Angular 19
- **Styling:** Tailwind CSS (Custom Typography, Glassmorphism, Component Themes)
- **Data Visualization:** Chart.js integration
- **Backend & Auth:** Supabase (PostgreSQL, OAuth)

## ✨ Core Features
- **Real-Time Processing:** Count words, characters (with/out spaces), sentences, paragraphs, and reading velocity instantly as you type inside your browser memory cache.
- **Word Frequencies:** Extract occurrences of top lexical items mapped natively to responsive, animated bar charts.
- **Secure Cloud Snapshots:** Save your entire historical analysis session to a heavily encrypted private Postgres cloud table via Row Level Security (RLS).
- **Universal Authentication:** Create an account natively or seamlessly **Continue with Google** via the OAuth provider implementation.
- **Dynamic Theming:** Seamless Dark Mode / Light Mode scaling tied directly into your database user profile to persist across browser sessions.

## ⚙️ Getting Started locally

1. **Clone the repository:**
   ```bash
   git clone https://github.com/cawle/TiroText.git
   cd TiroText
   ```
2. **Install dependencies:**
   ```bash
   npm install
   ```

   ```
3. **Run Development Server:**
   ```bash
   npm run dev
